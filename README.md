# Employee Attrition Analysis and Prediction

## Overview
The **Employee Attrition Analysis and Prediction** project is a machine learning application designed to help HR departments analyze employee data and predict the likelihood of an employee leaving the company. It features a robust predictive model and an interactive web dashboard built with Streamlit.

## Features
- **Data Overview**: View summary statistics and a preview of the dataset.
- **Exploratory Data Analysis (EDA)**: Interactive visual analysis of key factors influencing attrition (e.g., Age, Job Satisfaction).
- **Predict Attrition**: A user-friendly interface to input employee details and get real-time predictions on whether the employee is likely to stay or leave.

## Technologies Used
- **Python**
- **Machine Learning**: `scikit-learn` (Random Forest Classifier)
- **Data Manipulation**: `pandas`, `numpy`
- **Data Visualization**: `matplotlib`, `seaborn`
- **Web Framework**: `streamlit`

## Project Structure
- `app.py`: The main Streamlit web application script.
- `train.py`: Script to preprocess data, train the Random Forest model, and save the model/encoders.
- `Employee_Attrition_Analysis.ipynb`: Jupyter notebook for in-depth data exploration and model prototyping.
- `requirements.txt`: List of Python dependencies required to run the project.
- `model.pkl` & `encoders.pkl`: Serialized machine learning model and label encoders (generated by running `train.py`).

## Dataset
The project uses the `Employee-Attrition` dataset, which contains various employee metrics such as Age, Daily Rate, Distance From Home, Environment Satisfaction, Job Satisfaction, Work-Life Balance, Years at Company, and more.

## Installation and Setup

1. **Clone the repository:**
   ```bash
   git clone <your-repository-url>
   cd Employee-Attrition-Analysis-and-Prediction
   ```

2. **Create a virtual environment (optional but recommended):**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
   ```

3. **Install the required dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Train the model:**
   Run the training script to generate the `model.pkl` and `encoders.pkl` files.
   ```bash
   python train.py
   ```

5. **Run the Streamlit app:**
   ```bash
   python -m streamlit run app.py
   ```

6. **Access the application:**
   Open your web browser and navigate to `http://localhost:8501`.

## Usage
1. Start the app using the command mentioned above.
2. Use the **Navigation** sidebar to explore different sections: Data Overview, Exploratory Data Analysis, and Predict Attrition.
3. In the **Predict Attrition** section, enter the details of an employee to see the prediction of whether they are likely to leave the organization.
